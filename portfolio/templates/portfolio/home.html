In my last year at primary school, the first PC appeared and I was fascinated. I've also immersed myself in learning to code with an ongoing goal of becoming a full-stack web developer. Now in my 30's I still say that's what I want to be when I grow up!

This is partly a portfolio of my experience and learning, in particular the self-directed kind, which I don't have certificates for. And it is also a learning process, under constant development as I discover new technologies and gain new skills. I built it using Python's Django Web-Framework - currently there's more going on behind the scenes than there is out front!

If you're learning to code too, then check out my blog where I'll recommend lots of resources that have helped me. If you want to know if I can help you or your organisation then read on or get in touch!
